#  Dice Game

A simple and interactive Dice Game built using HTML, CSS, and JavaScript. This project generates random dice numbers for two players and determines the winner instantly on page refresh.

---

##  Overview

This project demonstrates basic JavaScript concepts like random number generation, DOM manipulation, and dynamic UI updates.

When the page is refreshed:
- Two dice are rolled randomly
- The results are displayed visually
- A winner is घोषित automatically

---

##  Features

-  Random dice roll for two players  
-  Automatic winner declaration  
-  Refresh to play again  
-  Simple and clean user interface  

---

##  Technologies Used

- HTML5 – Structure  
- CSS3 – Styling  
- JavaScript – Game logic & DOM manipulation  

---

##  How It Works

1. Page loads or refreshes  
2. JavaScript generates random numbers (1–6)  
3. Dice images update based on numbers  
4. Winner is displayed:
   - Player 1 wins  
   - Player 2 wins  
   - Draw
